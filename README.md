# node-express 练习
### 开始
1. 安装node 6.11
2. 安装cnpm （注：为了方便起见，node_modules以上传）
3. 安装本地 mongodb
```js
    启动并创建shopDemo数据库
```
4. 初始化（注：node_modules已存在）
```js
    cnpm i
```
5. 启动 
```js
  $  node app
```
